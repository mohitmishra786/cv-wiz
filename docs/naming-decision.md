## Recommendation: **MatchQuill** (matchquill.com)

MatchQuill encodes both halves of the product in one coined compound: **match** (the core engine — matching a job description to the user's profile) and **quill** (the writing output — ATS-optimized resumes and cover letters). It is two syllables, spells exactly as it sounds, has zero exact-match web presence, zero GitHub repositories, no npm package, no discoverable trademark filing, and — decisively — **matchquill.com returned "No match for domain" on a live WHOIS lookup on 2026-07-14**, the strongest available-signal short of a registrar cart. It shares no root with any of the six occupied names (no wizard/compiler/forge/loop/matcher-as-noun), and unlike nearly every candidate tested in this saturated category, no live career-tools product collides with it.

## VERIFY BEFORE PURCHASE — Human Checklist

Do all of these **before** announcing, printing, or building on the name. This naming space is being actively claimed (three near-identical domains were registered in the last 6 months), so do step 1 **immediately**.

1. **Registrar lookup & purchase:** Search `matchquill.com` at a real registrar (Cloudflare/Porkbun/Namecheap) and buy it same-day if free. Buy `quillfit.com` (top alternate) as a defensive/backup in the same session. WHOIS "no match" from 2026-07-14 can go stale in hours.
2. **USPTO TESS direct search:** Go to https://tmsearch.uspto.gov and run exact + phonetic searches for `matchquill`, `match quill`, and `quill` in International Classes 9, 35, and 42. My automated attempts were blocked (Trademarkia 403); only web-search-level clearance was performed. Consider a knockout search by a TM attorney before spending on brand assets.
3. **EUIPO / WIPO check** if you plan to sell outside the US: https://euipo.europa.eu/eSearch/ and https://branddb.wipo.int.
4. **X/Twitter handle:** Check and claim `@matchquill` directly at x.com — I could not verify handle availability by automation. Also claim Instagram/LinkedIn/YouTube handles.
5. **npm & GitHub org:** `npm org` / package name `matchquill` was 404 at check time — publish a placeholder package and register the `matchquill` GitHub org immediately after domain purchase.
6. **Chrome Web Store:** Search "MatchQuill" inside https://chromewebstore.google.com directly — I could only check via general web search (no store-internal query was possible). Confirm no extension exists before submitting yours.
7. **App-store sweep (optional):** Apple App Store / Google Play search for "MatchQuill" if a mobile app is ever planned.
